## Компиляция кода

Сначала необходимо установить **Python 3.12** (другие версии на данный момент не тестировались). Рекомендуется создать **виртуальное окружение Python** (например, `conda` или `venv`).

После входа в виртуальное окружение установите необходимые зависимости:

```cmd
pip install -r requirements.txt

После этого код будет готов к запуску.

При желании вы можете создать исполняемый файл с помощью следующей команды:

pip install pyinstaller & pyinstaller main.py

Аргументы

Опция	Длинная опция	Описание
-h	--help	Показать это справочное сообщение и выйти
-q QUERY	--query QUERY	Поисковый запрос (пример: mustafa_kemal_ataturk)
-t	--typeahead	Запуск API Typeahead
-f	--full-text-search	Запуск API полнотекстового поиска
-s	--search	Выполнить базовый поиск
-T	--tui	скоро будет доступно..
-l LIMIT	--limit LIMIT	Установить лимит количества результатов
-o OFFSET	--offset OFFSET	Значение смещения

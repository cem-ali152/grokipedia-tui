## Code kompilieren

Zuerst muss **Python 3.12** installiert sein (eine andere Version funktioniert möglicherweise nicht).
Erstellen Sie eine **Python-virtuelle Umgebung** (z. B. `conda` oder `venv`).

Nachdem Sie die virtuelle Umgebung aktiviert haben, installieren Sie die erforderlichen Abhängigkeiten:

...
| Option | Lange Option        | Beschreibung                                       |
| ----- | ------------------- | -------------------------------------------------- |
| `-h`  | `--help`            | Diese Hilfemeldung anzeigen und beenden           |
| `-q QUERY` | `--query QUERY` | Suchanfrage (Beispiel: `mustafa_kemal_ataturk`)   |
| `-t`  | `--typeahead`       | Typeahead-API ausführen                           |
| `-f`  | `--full-text-search`| Volltextsuch-API ausführen                        |
| `-s`  | `--search`          | Eine einfache Suche durchführen                  |
| `-T`  | `--tui`             | demnächst..                                      |
| `-l LIMIT` | `--limit LIMIT`| Ergebnislimit festlegen                          |
| `-o OFFSET`| `--offset OFFSET`| Offset-Wert                                     |
